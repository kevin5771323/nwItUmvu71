# 电影院管理系统 java78

## 项目概述

电影院管理系统是一个基于Java和SpringBoot，前端使用Vue构建的Web网站。该系统实现了前后端分离，具有完善的功能模块和简洁的用户界面。

## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows/Mac
- **开发环境**:
  - IDEA
  - JDK1.8
  - Maven3.6.1
  - Mysql
  - node14
  - navicat
- **技术栈**:
  - Java
  - Mysql
  - SpringBoot
  - Mybatis
  - Vue

## 功能模块

### 用户端前台

- 查看电影榜单
- 影片分类
- 购买电影票（包含选座，订单支付）
- 个人订单管理

### 管理员端后台

- 影片管理
- 影厅管理
- 订单管理
- 排片管理

## 系统角色

- 普通用户
- 管理员

## 运行环境

- 服务器：任何支持Java和Vue的服务器
- 数据库：Mysql
